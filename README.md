# SIG-SOBSE

Estructura inicial limpia y modular para el nuevo sistema de información geográfica.

## Carpetas principales
- frontend/: Código del cliente web (mapas, UI, etc.)
- backend/: Lógica del servidor, controladores, procesamiento
- apis/: Endpoints y servicios API
- assets/: Recursos estáticos (imágenes, íconos, estilos)
- docs/: Documentación técnica y de usuario
- _private_data/: Seeds y datos privados (respaldados)
- _BACKUP_20260409/: Backup de datos y seeds
